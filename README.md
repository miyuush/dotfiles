## Usage

### For Ubuntu

```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/miyuush/dotfiles/master/setup.sh)"
```

### [Fisher](https://github.com/jorgebucaran/fisher)

```sh
# Install Fisher
curl -sL https://git.io/fisher | source

# Install Fish plugin
fisher update
```
